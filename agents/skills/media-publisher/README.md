# 自媒体发布 Skill

## 功能描述
自动化发布内容到多个自媒体平台（微信公众号、知乎、小红书、抖音、百家号、搜狐号等）

## 能力清单
- 多平台账号管理
- 内容自动发布
- 平台规则适配
- 发布效果追踪
- Cookie 状态管理

## 使用方式
```markdown
请使用自媒体发布技能，帮我发布以下内容到[平台]：
标题：[标题内容]
正文：[正文内容]
配图：[图片URL或路径]
```

## 支持平台
- 微信公众号
- 知乎
- 小红书
- 抖音
- 百家号
- 搜狐号
- 微博

## 配置要求
- 平台账号 Cookie
- API 密钥（如需要）
- 发布配额
